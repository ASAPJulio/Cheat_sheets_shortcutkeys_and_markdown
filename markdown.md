# Cheat Sheet Markdown

## Titles

```markdown
# Titre de niveau 1
## Titre de niveau 2
### Titre de niveau 3
#### Titre de niveau 4
##### Titre de niveau 5
###### Titre de niveau 6
```

## Text with properties

```markdown
*Italics text*
_Italics text_

**Bold text**
__Bold text__

***Bold an italics text***
___Bold an italics text___
```
